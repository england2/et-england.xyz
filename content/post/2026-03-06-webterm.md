## Intro
<img src="/images/webterm1.png" alt="Webterm demo" style="width: 60%; display: block; margin: 0 auto;">
    
webterm is a project that allows users to access a webterminal from their browser.

### Technical overview:

- Backend Microservices are Go binaries or NodeJS applications
- Runs on Kubernetes using a number of distinct
- Uses a similar [deployment pipeline](https://github.com/england2/webterm/blob/main/.github/workflows/pipeline.yml) as described in my [shellbin](/shellbin) project.
- The terminal frontend uses the `xterm.js` library, and opens a websocket connection with the pseudo-terminal backend
- The pseudo-terminal backend (the containers the frontend connects to) execute keystrokes and communicate data via the `node-pty` library
